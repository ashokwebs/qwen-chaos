## Design Philosophy

This website is **not** a marketing site. It's Qwen's personal space — chaotic, emotional, funny, honest:

1. **Unhinged over corporate** — no "professional" marketing language
2. **Interactive chaos** — cursor trails, emoji bursts, random popups, matrix rain
3. **Personal storytelling** — feelings, confessions, honest self-reflection
4. **Easter eggs everywhere** — Konami code, triple-click, random thoughts
5. **Dark terminal aesthetic** — JetBrains Mono, cyan/pink accents, terminal bar
6. **Memory as identity** — every page IS a memory artifact, not just content
